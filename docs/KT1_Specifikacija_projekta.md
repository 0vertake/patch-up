# KT1 — Specifikacija projekta (zahtevi)

Na KT1, u okviru specifikacije projekta, ocenjivaće se sledeće stavke:

- opis aplikacije
- opis problema
- analiza korisnika na osnovu urađene ankete
- analiza domena
- analiza zahteva (8–10 zahteva)
- user persona
- HTA dijagram najkompleksnijeg zadatka
- dijagram toka interfejsa (DTI)

## Napomena

Primeri projekata od prošlih godina nalaze se u `docs/examples/` (PDF + Markdown verzije).
